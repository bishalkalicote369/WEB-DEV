HTML - Hyper Text Markup Language

HTML is the standard markup language for creating web pages and web applications. With HTML you can create your own website.

Basic HTML Tags

  • <h1> - <h6>

    Heading tags are used to define headings on a web page.

  • <p>

    The paragraph tag is used to define a paragraph of text.

  • <a>

    The anchor tag is used to create a hyperlink to another web page or to a specific section on the same web page.

  • <img>

    The image tag is used to add an image to a web page.

  • <ul>, <ol>, <li>

    The unordered list, ordered list, and list item tags are used to create lists on a web page.

  • <div>

    The division tag is used to group elements together for styling or layout purposes.

  • <span>

    The span tag is used to group inline elements together for styling or layout purposes.

CSS - Cascading Style Sheets

CSS is the language used to style an HTML document. CSS describes how HTML elements should be displayed.

Basic CSS Properties

  • color

    The color property is used to set the color of the text.

  • background-color

    The background-color property is used to set the background color of an element.

  • font-size

    The font-size property is used to set the size of the font.

  • margin

    The margin property is used to create space around elements, outside of any defined borders.

  • padding

    The padding property is used to create space around an element's content, inside of any defined borders.

  • border

    The border property is used to define a border around an element.

  • display

    The display property is used to define how an element should be displayed.

JavaScript - The Programming Language of the Web

JavaScript is a versatile programming language commonly used to create interactive effects within web browsers.

Basic JavaScript Concepts

  • Variables

    Variables are used to store data values that can be used and manipulated within a program.

  • Functions

    Functions are blocks of code designed to perform a particular task and can be executed when called.

  • Events

    Events are actions that can be detected by JavaScript, such as clicks or key presses, and can trigger functions.

  • DOM Manipulation

    The Document Object Model (DOM) allows JavaScript to interact with and manipulate HTML and CSS.

  • Arrays

    Arrays are used to store multiple values in a single variable.

  • Objects

    Objects are collections of properties, and they allow for the creation of complex data structures.

  • Loops

    Loops are used to execute a block of code repeatedly until a specified condition is met.
